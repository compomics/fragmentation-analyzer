FragmentationAnalyzer

FragmentationAnalyzer has been developed by Harald Barsnes (University of Bergen, Norway) under the guidance of Prof. Dr. Lennart Martens.

Project home page: http://fragmentation-analyzer.googlecode.com

FragmentationAnalyzer is based on the ms_lims project.

In addition code from the following projects is used:
  • Mascot dat file parsing is done using the MascotDatfile library
  • OMSSA parsing is done using omssa-parser
  • The platform independent graphical user interface is based on code from JGoodies
  • Advanced GUI components from SwingLabs SwingX
  • Plotting is done using JFreeChart
  • Spectrum visualization is done using the Gent utilites library
  • Correlation is calculated using Commons-Math
  • Exporting plots to SVG is done using Batik
